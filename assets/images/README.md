# Placeholder изображений

Эта папка предназначена для хранения изображений портфолио:

- `profile.jpg` - фото профиля
- `project-screenshots/` - скриншоты проектов
- `icons/` - иконки и логотипы
- `backgrounds/` - фоновые изображения

## Рекомендации по изображениям

- Используйте оптимизированные изображения (WebP, оптимизированный JPEG/PNG)
- Рекомендуемые размеры:
  - Фото профиля: 400x400px
  - Скриншоты проектов: 1200x800px
  - Иконки: 64x64px или SVG

## Инструменты для оптимизации
- TinyPNG - онлайн оптимизация
- Squoosh - веб-приложение от Google
- ImageOptim - для macOS
