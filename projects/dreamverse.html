Project Overview

Technical Architecture

System Components

Unreal Engine 5

Game client with UnrealMCP plugin. Handles rendering, physics, and player input. Communicates with backend via TCP socket.

MCP Server (Python)

Middleware layer for procedural generation. 21+ tools for creating structures, NPCs, terrain. Port 55557 TCP connection.

FastAPI Backend

REST API for game logic, authentication, Demiurge weapon generation. Port 8000 with OpenAPI docs.

Database Layer

SQLite for development, PostgreSQL planned for production. Stores players, weapons, NPCs, land plots.

Blockchain Layer

Polygon zkEVM smart contracts. ERC-721 land NFTs, ERC-6551 for NPC ownership (designed, not deployed).
