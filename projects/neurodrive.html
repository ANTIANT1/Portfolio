Project Overview

NeuroDrive is an automated ETL (Extract, Transform, Load) pipeline that intelligently processes documents from Google Drive, applies AI-powered categorization and analysis, and generates structured Markdown files for Obsidian knowledge base. Successfully processed 1000+ documents in production environment.

Key Features

Google Drive Integration

OAuth2 authentication with credentials.json. Automatic file collection with support for Docs, Sheets, PDFs. Date-based relationship detection.

OAuth2 Drive API v3 Python SDK

Smart Title Generation

AI-powered title generation based on document content. Replaces generic "Untitled" with meaningful names extracted from text.

NLP Content Analysis

AI Categorization

OpenAI GPT-4o analyzes content and assigns categories, extracts action items, generates summaries, identifies related projects.

GPT-4o Prompt Engineering

Obsidian Integration

Generates properly formatted Markdown files with YAML frontmatter, backlinks, tags, and metadata. Integrates into Obsidian vault structure.

Markdown YAML Obsidian

n8n Automation

Webhook integration for triggering downstream workflows. Sends summaries, action items, and metadata to automation platform.

n8n Webhooks HTTP

SQLite Storage

Local database for tracking processed files, caching metadata, storing analysis results. Prevents duplicate processing.

SQLite Python DB-API
