# 💼 JETZWORK - Децентрализованная B2B платформа для фриланса

> **Статус:** MVP в разработке | **Дата запуска:** 2025 | **Прогресс:** 70%

## 📖 Описание

JETZWORK — это децентрализованная B2B-платформа, соединяющая независимых самозанятых специалистов с бизнесами для выполнения краткосрочных работ. Идеологический антипод традиционных агентств временного трудоустройства (Zeitarbeit), предоставляющий исполнителям полный контроль над ставкой, графиком и выбором заказов.

## 🎯 Проблема и решение

### Проблема
- **Zeitarbeit агентства** забирают 30-50% от ставки исполнителя
- Исполнители не имеют контроля над условиями работы
- Долгий и бюрократический процесс найма
- Отсутствие прозрачности в оплате

### Решение
JETZWORK предоставляет:
1. **Прямую связь** между исполнителем и заказчиком
2. **Полный контроль** над ставкой и условиями
3. **Мгновенный матчинг** через умный алгоритм
4. **Прозрачность** через блокчейн и репутационную систему

## ✨ Ключевые особенности

### 🎭 Гостевой режим для исполнителей
- Просмотр доступных заказов без регистрации
- Фильтры по локации, ставке, категории
- Конверсионный flow для регистрации
- Геймификация процесса подключения

### 🏅 Система верификации и доверия
**3 уровня верификации:**
- **Bronze** - базовая регистрация (email, телефон)
- **Silver** - документы (паспорт, самозанятость)
- **Gold** - полная верификация (видео-звонок, рекомендации)

**Индекс надежности (Trust Index):**
- Автоматический расчет на основе истории
- Учет завершенных заказов, отзывов, времени на платформе
- Влияет на видимость в поиске

### 🤝 Умное сопоставление (Smart Matching)
- ML-алгоритм подбора исполнителей
- Учет навыков, рейтинга, локации
- Предсказание успешности выполнения
- Автоматические рекомендации

### ⏱️ Учет рабочего времени (Arbeitszeiterfassung)
- Встроенный таймер с GPS-подтверждением
- Фото-отчеты с геолокацией
- Автоматическое формирование табелей
- Соответствие немецкому законодательству

### ⚖️ Система разрешения споров
- Медиация через платформу
- Escrow система для платежей
- Голосование сообщества
- Страхование сделок

## 🛠 Технологический стек

### Mobile App
- **Flutter 3.x** - кросс-платформенная разработка
- **Dart** - язык программирования
- **Provider/Riverpod** - state management
- **Google Maps SDK** - карты и геолокация
- **Firebase Auth** - аутентификация

### Backend (BaaS)
- **Firebase** - Backend-as-a-Service
- **Firestore** - NoSQL база данных (real-time)
- **Cloud Functions** (TypeScript) - серверная логика
- **Firebase Storage** - хранилище файлов
- **FCM** - push уведомления

### Additional Services
- **Google Maps Platform API** - локация и маршруты
- **Stripe/PayPal** - платежи
- **SendGrid** - email уведомления
- **Twilio** - SMS верификация

## 💡 Примеры использования

### 1. Для исполнителя
```
1. Открыть app → Гостевой режим
2. Увидеть заказ "Официант на 3 часа"
3. Зарегистрироваться через телефон
4. Подать заявку на заказ
5. Получить подтверждение
6. Начать работу с GPS-таймером
7. Получить оплату после завершения
```

### 2. Для бизнеса
```
1. Зарегистрироваться как Business
2. Создать заказ "Нужен водитель"
3. Указать ставку, время, локацию
4. Получить предложения от исполнителей
5. Выбрать подходящего
6. Подтвердить выполнение работы
7. Оставить отзыв
```
