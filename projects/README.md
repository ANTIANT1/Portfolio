# Примечания по проектам

Этот файл содержит общую информацию о проектах в портфолио.

## Структура папки projects

Каждый проект должен быть размещен в отдельной подпапке со следующей структурой:

```
project-name/
├── README.md          # Описание проекта
├── src/              # Исходный код
├── assets/           # Ресурсы проекта
└── docs/             # Документация
```

## Рекомендации

1. Используйте понятные названия папок (например: `web-calculator`, `todo-app`)
2. Всегда добавляйте README.md с описанием проекта
3. Документируйте код на русском языке
4. Добавляйте скриншоты или демо проекта

## Категории проектов

- **Web-разработка** - веб-приложения и сайты
- **Desktop** - десктопные приложения
- **Mobile** - мобильные приложения
- **Scripts** - утилиты и скрипты
- **Игры** - игровые проекты
- **Другое** - прочие проекты
